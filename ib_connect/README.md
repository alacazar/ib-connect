# IB Connect

A multi-skill project for Interactive Brokers integration, providing modular tools for querying, downloading, and managing IB data.

## Skills

- **ib-query**: CLI tool for querying contract details.
- [Future: ib-download, ib-account, etc.]

## Shared Code

Common utilities in `shared/` for reuse across skills.

## Installation

Clone the repo, install dependencies:

```bash
pip install -e .
```

See each skill's `SKILL.md` for details.
